# agent-hygiene

<img src="./agent-hygiene.png" width="500"  />

**LeanMaxxing** is a public, competitive leaderboard where developers flex how efficiently their agent systems run- doing more with less, minimizing token usage, and eliminating waste. 

**Agent Hygiene** is the underlying system that makes it meaningful: a structured score that evaluates how clean, well-tuned, and balanced an agent setup is across flow, redundancy, and structure. Together, LeanMaxxing is the visible outcome, and Agent Hygiene is the disciplined practice behind it.
